# RER 热量计算公式说明

静止能量需求 (Resting Energy Requirement, RER) 是计算宠物每日热量需求的基础。

## 基础公式

```
RER = 70 × (体重kg)^0.75
```

例如：
- 5kg 猫：RER = 70 × 5^0.75 = 70 × 3.34 = 234 kcal/day
- 15kg 狗：RER = 70 × 15^0.75 = 70 × 7.62 = 533 kcal/day
- 30kg 狗：RER = 70 × 30^0.75 = 70 × 12.82 = 897 kcal/day

## 特殊状况调整

| 状况 | 系数调整 | 说明 |
|------|----------|------|
| 肥胖/减重 | × 0.8 | 减少热量 |
| 怀孕后期 | × 2.0 | 增加热量 |
| 哺乳期 | × 2-4 | 根据幼崽数量 |
| 术后恢复 | × 1.5 | 临时增加 |
| 神经性食欲不振 | × 1.2 | 临时增加 |

## 完整计算示例

**示例 1**: 15kg 成年犬，中等活动

```
RER = 70 × 15^0.75 = 533 kcal
每日热量 = 533 × 2.0 = 1066 kcal
```

**示例 2**: 4kg 室内绝育猫

```
RER = 70 × 4^0.75 = 70 × 2.82 = 197 kcal
每日热量 = 197 × 1.2 = 237 kcal
```

**示例 3**: 5kg 幼犬（4月龄）

```
RER = 70 × 5^0.75 = 234 kcal
每日热量 = 234 × 3.5 = 819 kcal
```

## 营养素分配

基于每日热量，分配三大营养素：

### 犬 (AAFCO 标准)

| 营养素 | 最小 | 推荐范围 | 说明 |
|--------|------|----------|------|
| 蛋白质 | 18% | 25% ± 7% | 生长期需更高 |
| 脂肪 | 5.5% | 15% ± 5% | |
| 碳水化合物 | - | 50% ± 10% | 无最小要求 |

### 猫 (AAFCO 标准)

| 营养素 | 最小 | 推荐范围 | 说明 |
|--------|------|----------|------|
| 蛋白质 | 26% | 35% ± 5% | 纯肉食动物 |
| 脂肪 | 9% | 20% ± 5% | |
| 碳水化合物 | - | 30% ± 10% | 需求较低 |

### 营养素热量换算

- 蛋白质: 4 kcal/g
- 脂肪: 9 kcal/g
- 碳水化合物: 4 kcal/g

**示例计算**（1000 kcal 日粮，犬标准）:
- 蛋白质: 1000 × 0.25 / 4 = 62.5 g
- 脂肪: 1000 × 0.15 / 9 = 16.7 g
- 碳水: 1000 × 0.50 / 4 = 125 g
