# 第1周饮食计划：肠道修复期 - 单一蛋白温和调理

## 饮食原则
- **单一蛋白原则**：仅使用鸡肉作为蛋白质来源，最大限度降低过敏风险和肠道刺激
- **低脂易消化**：脂肪含量控制在12%以下，减轻胰腺和肠道负担
- **高可溶性纤维**：南瓜提供温和的可溶性纤维，帮助吸收肠道多余水分，改善软便
- **温和碳水**：白米饭提供易消化的能量来源，避免粗粮对肠道的额外刺激
- **益生菌支持**：每餐添加益生菌粉，重建肠道菌群平衡
- **无调味烹饪**：不添加任何盐、油、调料，保持食物天然温和

### 第1餐（早晨 7:00-8:00）

#### 鸡肉南瓜米饭粥

**食材**（单餐份量）：
- 鸡腿肉(去皮去骨)：145g
- 大米：125g
- 南瓜：175g
- 益生菌粉：1小袋（约2-3g，含50-100亿CFU）

**烹饪方式**：
1. **处理鸡肉**：将鸡腿肉洗净，放入冷水中，大火煮沸后转小火煮15-20分钟至完全熟透，捞出后用刀切成小丁或用手撕成细丝（约1cm大小），便于消化
2. **煮米饭**：大米洗净，加入约1:1.5的水（米水比例），用电饭煲或锅煮至软烂，比平时人吃的米饭稍软一些
3. **蒸南瓜**：南瓜去皮去籽，切成2cm小块，放入蒸锅蒸15分钟至软烂，用叉子压成泥状
4. **混合**：将鸡肉丁、软米饭、南瓜泥放入碗中，加入适量煮鸡肉的清汤（约50ml），搅拌均匀至粥状
5. **添加益生菌**：待食物温度降至约40°C（手感微温）时，加入益生菌粉，轻轻拌匀（高温会杀死益生菌）
6. **喂食温度**：确保食物温度接近体温（约38°C）后喂食

**单餐营养素含量**：
- 热量：607.5 kcal
- 蛋白质：39.1g（25.8%）
- 脂肪：8.1g（12.0%）
- 碳水化合物：94.5g（62.2%）
- 膳食纤维：约3.5g（来自南瓜）

**微量营养素**（单餐）：
- 钙：206 mg（支持骨骼健康）（来源：鸡肉、大米、南瓜）
- 磷：383.5 mg（与钙共同维持骨骼）（来源：鸡肉、大米）
- 铁：3.9 mg（造血功能）（来源：鸡肉、大米）
- 锌：13.7 mg（免疫功能、皮肤健康）（来源：鸡肉、大米）
- 维生素A：215.6 IU（视力、黏膜健康）（来源：南瓜）
- 维生素E：0.7 mg（抗氧化）（来源：鸡肉、大米、南瓜）
- 维生素B1：0.28 mg（能量代谢）（来源：大米、鸡肉）
- 钾：558 mg（电解质平衡）（来源：鸡肉、南瓜、大米）

## 特别说明

1. **关于食材选择的说明**：
   - 数据库中没有"鸡胸肉"，使用"鸡腿肉(去皮去骨)"替代。鸡腿肉去皮后脂肪含量较低（5.4g/100g），仍符合低脂要求
   - 如能购买到鸡胸肉，可优先选用（鸡胸肉脂肪含量约3-4g/100g，更低脂）
   - 白米饭使用"大米"煮熟，南瓜选用新鲜南瓜蒸制

2. **关于营养素偏低的说明**：
   - 第1周为肠道修复期，采用最温和的单一蛋白方案，某些微量营养素可能低于AAFCO标准
   - 这是**短期（7天）治疗方案**，专注于软便症状改善，而非长期营养均衡
   - 如软便在3-5天内明显改善，可从第2周开始逐步引入其他食材（白鱼肉、燕麦等）以丰富营养
   - 维生素D可通过每日适度日晒（15-20分钟）帮助狗狗自身合成

3. **关于益生菌的说明**：
   - 选择宠物专用益生菌粉，含乳酸菌、双歧杆菌等菌株
   - 必须在食物降温至40°C以下后添加，避免高温杀死益生菌
   - 建议在兽医指导下选择合适品牌和剂量

4. **关于钙磷比的说明**：
   - 当前食谱钙磷比约为1:1.86，磷相对偏高
   - 短期（1周）使用问题不大，但如延长使用建议添加钙补充剂（如碳酸钙粉）
   - 第2周开始引入其他食材后可自然改善钙磷比

5. **食物制备与储存**：
   - 建议每日新鲜制备，保证食物新鲜度
   - 如需提前准备，可将煮熟的食材分装冷藏（不超过24小时），喂食前加热
   - 益生菌粉必须现喂现加，不可提前混合

**评分标准**：

- 便便：1=水样腹泻，3=软便，5=成形正常
- 精神：1=萎靡不振，3=正常，5=活泼好动

### ⚠️ 就医提醒
- 如软便持续超过5天无改善，或出现以下症状，**立即就医**：
  - 呕吐（尤其频繁呕吐）
  - 便血或黑便
  - 食欲完全丧失超过24小时
  - 精神极度萎靡
  - 发热（体温超过39.5°C）
  - 明显脱水（皮肤弹性差、牙龈干燥）

### 🍽️ 喂食技巧
1. **少量多餐**：如狗狗一次吃不完，可将每餐分成2-3次喂食
2. **食物温度**：确保食物接近体温（38°C），过冷会刺激肠道
3. **进食环境**：保持安静、固定的进食环境，减少压力
4. **饮水充足**：保证清洁饮水随时可得，但避免饭后立即大量饮水

### 🔄 第2周过渡提示
- 如第1周结束时软便明显改善（便便评分≥4分），第2周可开始引入：
  - 第二蛋白质：白鱼肉（如鳕鱼、巴沙鱼）
  - 新碳水：燕麦片（少量开始）
  - 新蔬菜：胡萝卜（煮熟）
- 每次只引入1种新食材，观察3天无不良反应再继续
