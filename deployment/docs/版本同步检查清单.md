## ⚠️ 版本升级注意事项

### 升级 Node.js

1. **更新本地环境**
   ```bash
   # 使用 nvm
   nvm install 24
   nvm use 24
   ```

2. **更新前端 Dockerfile**
   ```dockerfile
   FROM node:24-alpine AS builder
   ```

3. **更新 GitHub Actions**
   ```yaml
   env:
     NODE_VERSION: '24'
   ```

4. **测试构建**
   ```bash
   cd frontend/web-app
   npm install
   npm run build
   npx cap sync android
   ```

---

### 升级 PostgreSQL

1. **备份数据**
   ```bash
   docker exec pet-food-postgres pg_dump -U petfood petfood > backup.sql
   ```

2. **更新 docker-compose.prod.yml**
   ```yaml
   postgres:
     image: postgres:17-alpine
   ```

3. **重新部署**
   ```bash
   docker compose down
   docker volume rm pet-food_postgres_data
   docker compose up -d
   ```

4. **恢复数据**
   ```bash
   cat backup.sql | docker exec -i pet-food-postgres psql -U petfood petfood
   ```
